Trabalhando com Listas

Listas Ordenadas

Para se criar uma lista ordenada usa-se a tag <ol> e aninha-se a tag <li> para numerar e determinar quantos items constituem a lista:

  1. Acordar
  2. Correr 5km
  3. Tomar banho
  4. Comer
  5. Estudar

OBS: Usa-se o atributo (parâmetro) type=" " na tag de abertura da lista ordenada para determinar a forma em que os números dos items da lista serão exibidos e o parâmetro start =" ": que será responsável em determinar em que valor a contagem dos items iniciará.

  1. Acordar
  2. Correr 5km
  3. Tomar banho
  4. Comer
  5. Estudar

Listas não Ordenadas

Para se criar uma lista ordenada usa-se a tag <ul> e aninha-se a tag <li> para determinar quantos items constituem a lista:

  • Fuba
  • Pão
  • Manteiga
  • Múkua
  • Kitaba

OBS: Usa-se o atributo (parâmetro) type=" " na tag de abertura da lista não ordenada para determinar a forma em que os marcadores de item serão exibidos.

  • Fuba
  • Pão
  • Manteiga
  • Múkua
  • Kitaba

Lista de Definições

Para se criar uma lista de definições usa-se a tag <dl> e aninham-se as tags <dt> que é o item a ser definido e <dd> que é a definição do item :

Toyota
Marca de carro japonesa.
Armani
Marca de roupa italia.
