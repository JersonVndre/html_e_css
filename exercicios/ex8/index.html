Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase, temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>.

E no outro parágrafo temos outro texto marcado no final.

Texto Grande e pequeno

Estámos criando um texto grande com a tag <big> (tag obsoleta) e um texto pequeno com a tag <small> nesse parágrafo.

Texto excluído

Podemos marcar um texto como excluído usando a tag <del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e para indicar que ele foi adicionado depois e também para sublinhar.

Texto Sobrescrito

Para inserir coisas como 32 + 25.

Texto Subescrito

Para inserir coisas como H2O.

Outras Formatações

Exemplo de código em PortugolStudio

Podemos escrever trechos de código usando a tag <code>:

inteiro n escreva("Digite um número:") leia(n) se(n % 2 == 0){ escreva("O número ,n, é um PAR.") }senao{ escreva("O número ,n, é um ÍMPAR.") } escreva("Fim do programa!")

Indentando o código

Indentá-mos o código usando a tag <code> aninhada dentro da tag <pre>:

       
            inteiro n
            escreva("Digite um número:")
            leia(n)
            se(n % 2 == 0){
                    escreva("O número ,n, é um PAR.")
            }senao{
                escreva("O número ,n, é um ÍMPAR.")
            }
            escreva("Fim do programa!")

Citações simples

Podemos criar citações simples usando a tag <q>:

Como dizia o meu pai: Deus não gosta de burro.

Citações completas

Criá-mos citações completas usando a tag <blockquote>:

Segundo Chris Logan, em seu livro PTSD para Leigos:

Traumas que se pareciam insuperáveis foram superados, mas nem todos os traumas se podem superar!

Abreviações

Podemos escrever siglas e mostrar o seu significado usando a tag <abbr>:

Estou estudando HTML e CSS. Estou amando.
